# Wildfire Detection using CNN Ensemble

This project builds a deep learning system to detect wildfires from satellite and non-satellite images using Convolutional Neural Networks (CNNs) and an ensemble learning approach.

# Key Features
- CNN-based image classification
- Ensemble model to improve prediction robustness
- Achieved 92% classification accuracy
- Data preprocessing including normalization and augmentation

# Approach
- Trained multiple CNN models on wildfire image datasets
- Combined predictions using a meta-learning approach
- Evaluated performance using confusion matrix and validation metrics

# Results
- Accuracy: 92%
- Improved robustness compared to single model
- Reduced false positives through ensemble method

# Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib

# How to Run
1. Clone the repository  
2. Install dependencies  
3. Run the notebook: `WildFire_Detection.ipynb`
